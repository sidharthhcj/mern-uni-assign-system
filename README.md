# MERN University Assignment System

## 📌 Overview

The **MERN University Assignment System** is a web-based application designed to help universities manage assignment submissions efficiently.
It allows students to upload assignments and enables administrators/teachers to view and manage submitted files.

This project is built using the **MERN Stack (MongoDB, Express.js, React.js, Node.js)**.

---

## 🚀 Features

* Student login system
* Upload assignments with files
* Store assignment details in MongoDB
* View uploaded assignments on dashboard
* Admin/teacher can access submitted assignments
* File storage using server upload folder

## ⚙️ Installation

### 1️⃣ Clone the repository

```
git clone https://github.com/sidharthhcj/mern-uni-assign-system.git
```

### 2️⃣ Install backend dependencies

```
cd server
npm install
```

### 3️⃣ Install frontend dependencies

```
cd frontend
npm install
```

### 4️⃣ Start backend server

```
npm start
```

### 5️⃣ Start frontend

```
npm run dev
```
